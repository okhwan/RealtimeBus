실시간 버스 정보 시스템

이 프로그램은

이 프로그램은

현재 버스 정류장으로 오는 버스들과 해당 버스들의 노선을 알려주는 것과 달리 목적지 까지 무슨 버스를 타야하는지 실시간 정보를 통해 사용자가 현재 타야할 버스를 알려줍니다.

편리함.

더 이상 해당 버스가 어디 까지 가고 어떤 버스는 몇 시에 도착하는 지 알 필요 없이 당장 당신에게 진짜 필요한 버스 정보만 보여줍니다.

Redis를 사용한 캐시 및 세션을 이용하여 기존 정보를 활용할 수 있고 여러 서버를 둘 경우 당신의 로그인 정보는 중간에 어떤 서버가 고장나도 로그인 정보는 유지됩니다. MongoDB를 통해 로그 데이터 기록으로 어느 화면을 보였고 해당 사용자가 어떤 기능들을 활용 했는지 알 수 있습니다.
